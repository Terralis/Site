O que é o Terrális?

O Terrális é um projeto desenvolvido por alunos do curso técnico em Informática do Instituto Federal de São Paulo. ...
O projeto visa proporcionar um vasto banco de questões para auxiliar estudantes, com foco no primeiro ano do ensino médio, a estudarem para testes escolares e/ou vestibulares.

Quais são as funcionalidade do Terrális?

A plataforma do Terrális fornece materiais de apoio para revisar os conteúdos das Ciências da Natureza, além dos ...questionários para fixação destes assuntos.
O Terrális também conta com gráficos de desempenho para acompanhar a evolução de cada usuário.

Como funciona o Terrális?

O Terrális fornece material de apoio?

Sim. Para cada tópico de disciplina foram selecionados e elaborados videoaulas e resumos abordando o conteúdo ..., para ajudar o usuário a relembrar os devidos asssuntos antes de responder aos questionário.

Preciso pagar para usar o Terrális?

Não. O único requisito para utilizar a plataforma é o cadastramento, que permite acesso à todas as funcionalidades do sistema.
